8 Trading Strategies

EMA/SMA crossover, momentum breakout, scalping, trend following, mean reversion, RSI reversal, and multi-strategy fallback.

9-Dimension Confluence

Level proximity, trend alignment, momentum, BOS, order blocks, structure, session ORB, timing — scored 0-100.

🛡️

Adaptive Risk Management

%R position sizing, ATR-based SL/TP, daily/weekly/total circuit breakers, recovery mode. Account-tier adaptive.

🧠

Self-Improving

Deterministic reflection loop evolves strategy one variable at a time. Optional Hermes LLM for complex reasoning.

🌐

Exchange-Agnostic

ccxt supports 105+ exchanges. Pre-configured for Binance, Bybit, OKX, Kraken. Paper trading with synthetic data.

📡

Discord Alerts

Rich embed notifications for trades, signals, health, reflections, daily reports. Webhook + bot API support.

🔍

Regime Detection

5 market regimes (trending strong/weak, ranging, volatile, reversal) with confidence scoring and strategy recommendation.

💧

Liquidity Analysis

Stop hunt zones, fair value gaps, volume nodes, liquidity bias. Prevents entering into liquidity sweep zones.

📈

Position Management

Dynamic breakeven, trailing stops, profit profiling (partial closes at 1R/2R/3R). Local SL/TP monitoring.

Quick Start

Up and running in under a minute.

# Clone the repo
git clone https://github.com/artifact-opensource/sita.git
cd sita

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install
pip install -e .

# Configure
cp .env.example .env
# Edit .env with your exchange API keys

# Paper trading (safe default)
python3 -m sita run

# Live trading (when ready)
export SITA_TRADING_MODE=live
export SITA_I_ACCEPT_RISK=true
python3 -m sita run
